# Java Learning Journey: De la Lógica a la Estructura 🚀

Este repositorio recopila mis primeros pasos en el desarrollo de software con Java durante el primer año de DAW. Contiene ejercicios fundamentales que fueron la base de mi formación en programación lógica y estructurada.

## 📈 Evolución del Aprendizaje

He estructurado este repositorio para reflejar cómo mi código ha ido ganando en complejidad, modularidad y orden a medida que avanzaba en mis estudios:

### 1. Reto 1: Análisis de Ventas (Lógica de Control)
* **Enfoque:** Introducción a la algoritmia básica.
* **Competencias demostradas:** Gestión de flujos de control (`if/else`, `while`), acumuladores, contadores y clasificación de datos mediante reglas de negocio simples.

### 2. Reto 2: Censo de Habitantes (Modularidad y Estructuras)
* **Enfoque:** Abstracción de problemas complejos.
* **Competencias demostradas:** * **Modularidad:** Separación de la lógica en métodos (`numeroCalles`, `recogerDatos`, `resumenDatos`) para evitar un `main` sobrecargado.
    * **Estructuras de datos:** Gestión de datos relacionales mediante el uso de **arrays paralelos** (relación entre calles, portales y habitantes).
    * **Pasaje de parámetros:** Aprendizaje sobre cómo compartir información entre distintos métodos.
